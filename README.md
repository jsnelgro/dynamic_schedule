README
================================

<h2>Installing:</h2>
<ul>
	<li>Create folder in your Drupal installation path at:<code>sites/all/modules/dynamic_schedule</code></li>
	<li>Go to <code>Admin/build/modules</code> on your Drupal site</li>
	<li>Check "install" next to Dynamic Schedule</li>
	<li>Click "Save Configuration"</li>
</ul>

<h2>Usage:</h2>
<ul>
	<li>Add a scheduler block to the page you would like to display the schedule on</li>
	<li>Go to Create Content</li>
	<li>Create a new "Available Times" piece of content</li>
	<li>In the content creation form, name your schedule, select the page you want the times displayed on, then fill out your opening and closing times for each day</li>
</ul>
